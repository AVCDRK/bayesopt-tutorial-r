Introduction

Optimization of function \(f\) is finding an input value \(\mathbf{x}_*\) which minimizes (or maximizes) the output value:

\[ \mathbf{x}_* = \underset{\mathbf{x}}{\arg\min}~f(\mathbf{x}) \]

In this tutorial we will optimize \(f(x) = (6(x-2)^2)~\text{sin}(12x-4)\)1, which looks like this when \(x \in [0, 1]\):

The ideal scenario is that \(f\) is known, has a closed, analytical form, and is differentiable – which would enable us to use gradient descent-based methods. For example, here’s how we might optimize it with Adam(Kingma and Ba 2014) in TensorFlow(Allaire and Tang 2019):


library(tensorflow)
sess = tf$Session()

x <- tf$Variable(0.0, trainable = TRUE)
f <- function(x) (6 * x - 2)**2 * tf$sin(12 * x - 4)

adam <- tf$train$AdamOptimizer(learning_rate = 0.3)
opt <- adam$minimize(f(x), var_list = x)

sess$run(tf$global_variables_initializer())

for (i in 1:20) sess$run(opt)

Further reading

  • (Snoek, Larochelle, and Adams 2012) explains how BayesOpt may be used for automatic parameter tuning in machine learning
  • (Frazier 2018) is a more in-depth, conceptual tutorial
  • (Shahriari et al. 2016) provides a comprehensive review of the algorithm and its applications
  • (Letham et al. 2018) shows how Facebook uses BayesOpt to find next set of parameter values to evaluate with online experiments (A/B tests)
  • (Görtler, Kehlbeck, and Deussen 2019) is a visual exploration of Gaussian processes
